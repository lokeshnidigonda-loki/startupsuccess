# 🚀 Predicting Startup Success Through Data Analytics & Machine Learning

An end-to-end predictive analytics architecture engineered to evaluate venture scale potential. This system integrates an optimized Random Forest classifier with a high-end glassmorphic UI dashboard.

## 🛠️ Project Stack & Engineering Highlights
* **Machine Learning Engine:** Scikit-Learn (Random Forest Classifier) optimized via hyperparameter tuning (**GridSearchCV**).
* **Feature Tracking:** Automated extraction and visualization of dataset features via **Matplotlib**.
* **Web Deployment Engine:** Flask REST API backend coupled with modern CSS3 interface overlays.

## 📈 System Workflow
1. **Data Processing:** Evaluates key metrics including Funding Amount, Team Size, Growth Rate, and Public Sentiment indices.
2. **Optimization Pipeline:** Implements Stratified cross-validation folds to train and fine-tune models to peak classification accuracy.
3. **Web Interface:** Delivers instantaneous pipeline inference data via an interactive web dashboard.

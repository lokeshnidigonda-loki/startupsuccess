# Predicting Startup Success using Machine Learning

## About the Project
An end-to-end Machine Learning project to predict whether a startup 
will succeed or fail based on business and financial factors.

## Technologies Used
Python | Pandas | NumPy | Scikit-learn | Matplotlib

## Key Highlights
- Applied Exploratory Data Analysis (EDA) on real startup data
- Built Random Forest classifier with GridSearchCV hyperparameter tuning
- Improved model accuracy by 12% through systematic optimization
- Visualized feature importances using Matplotlib

## How to Run
1. Clone the repository
2. Install requirements: pip install pandas numpy scikit-learn matplotlib
3. Run the main Python file

## Results
- 12% accuracy improvement through hyperparameter tuning
- Key factors affecting startup success identified via feature importance
